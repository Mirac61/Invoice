# Invoice Frontend

React + TypeScript + Vite, spricht mit der [Backend-API](../backend/README.md).

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Dev-Server mit HMR
- `npm run build` — Typecheck + Production-Build
- `npm run lint` — Oxlint
- `npm run preview` — Production-Build lokal ansehen
